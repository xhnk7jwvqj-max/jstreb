# Queen Size Floor Bed Frame Plan

## DIY Woodworking Project

A minimalist Japanese-inspired floor bed frame (platform bed) for a queen size mattress (60" x 80").

---

## Overview

This floor bed frame sits low to the ground with a clean, modern aesthetic. It features a slatted base for mattress ventilation, sturdy construction, and a slight overhang around the perimeter for a floating appearance.

**Finished Dimensions:**
- Overall: 66" W x 86" L x 8" H
- Mattress Area: 60" W x 80" L
- Overhang: 3" on all sides

**Skill Level:** Beginner to Intermediate
**Build Time:** 4-6 hours

## Assembly Instructions

### Step 1: Prepare All Lumber

1. Cut all pieces according to the cut list
2. Sand all pieces with 120-grit, then 220-grit sandpaper
3. Round over sharp edges with sandpaper or router
4. Label each piece with painter's tape

### Step 2: Build the Frame

    ┌───────────
    │  ┌─────

**Corner Assembly Process:**

1. Position the corner block inside the corner, flush with the top of the rails
2. Pre-drill two holes through each rail into the corner block
3. Apply wood glue to contact surfaces
4. Drive lag screws with washers through rails into corner blocks
5. Repeat for all four corners
6. Check frame for square by measuring diagonals (should be equal)

### Step 3: Install Slat Supports

```
Cross Section View

    ┌───────────────────────────────────────────────────────┐ ← 2x8 Side Rail
    │                                                       │
    │   ┌─────────────────────────────────────────────┐    │
    │   │ ← 2x4 Slat Support (mounted 1.5" from top) │    │
    │   └─────────────────────────────────────────────┘    │
    └───────────────────────────────────────────────────────┘

Side View - Support Placement

    ←─────────────────── 80" ───────────────────→
    ┌──────────────────────────────────────────────┐
    │  ○     ○     ○     ○     ○     ○     ○     ○ │ ← Screws every 10"
    └──────────────────────────────────────────────┘
```

1. Measure 1.5" down from the top inside edge of each side rail
2. Mark a level line along the entire length
3. Position the 80" slat supports along this line (centered, 3" from each end)
4. Pre-drill and countersink holes every 10"
5. Apply wood glue and secure with 3" screws
6. Repeat for both sides

### Step 4: Install Center Support Beam

```
Top View - Center Beam Placement

    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │  ○────────────────○────────────────○           │
    │     ↑ Leg          ↑ Leg           ↑ Leg       │
    │                                                 │
    │              ← 30" →    ← 30" →                │
    │                                                 │
    │  ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ← Center Beam  │
    │                                                 │
    └─────────────────────────────────────────────────┘
```

1. Attach three 4.5" legs to the center beam at: both ends and center
2. Position the center beam assembly at the midpoint of the frame (30" from each side rail)
3. The top of the center beam should be flush with the slat supports
4. Secure the center beam to head and foot rails with angle brackets or toe-screwed screws

### Step 5: Install Bed Slats

```
Top View - Slat Layout

    ┌─────────────────────────────────────────────────┐
    │ ═══════════════════════════════════════════════ │ ← Slat 1 (flush with head)
    │                                                 │
    │ ═══════════════════════════════════════════════ │ ← Slat 2
    │                     ↕ ~5.7" gap                 │
    │ ═══════════════════════════════════════════════ │ ← Slat 3
    │                                                 │
    │         (continue pattern - 14 slats)          │
    │                                                 │
    │ ═══════════════════════════════════════════════ │ ← Slat 14 (flush with foot)
    └─────────────────────────────────────────────────┘

    ← 60" slat length →
```

**Slat Spacing Calculation:**
- Interior length: 80"
- 14 slats x 3.5" width = 49"
- Remaining space: 80" - 49" = 31"
- 13 gaps between slats
- Gap size: 31" ÷ 13 ≈ 2.4" (adjust as needed, 2-3" is ideal)

**Installation:**
1. Start at the head end, position first slat flush with interior edge
2. Use a 2.4" spacer block for consistent gaps
3. Pre-drill two holes at each end and two at center support
4. Secure each slat with 2" screws (6 screws per slat)
5. Do not over-tighten - slats should be snug but not bowed

### Step 6: Finishing

1. **Fill holes:** Use wood filler on any visible screw holes, let dry, sand smooth
2. **Final sand:** Light sanding with 220-grit over entire frame
3. **Clean:** Wipe down with tack cloth to remove dust
4. **Stain (optional):** Apply stain following manufacturer's directions
5. **Seal:** Apply 2-3 coats of polyurethane, sanding lightly between coats
6. **Felt pads:** Attach felt furniture pads to bottom of frame and center beam legs

## Tips for Success

### Wood Selection
- Choose straight lumber without warping, twisting, or large knots
- Kiln-dried lumber prevents future warping
- For a premium look, consider oak, walnut, or maple

### Mattress Ventilation
- The slat spacing allows air circulation under the mattress
- This prevents moisture buildup and extends mattress life
- Consider a breathable mattress protector

### Floor Protection
- Use felt pads to prevent floor scratching
- Consider rubber bumpers for hard floors
- Check periodically that frame hasn't shifted

### Weight Capacity
- This design supports 800+ lbs when properly constructed
- Ensure all joints are tight and lag screws are fully seated

---

## Customization Options

1. **Headboard:** Add a 2x10 or plywood headboard panel
2. **Stain colors:** Natural, walnut, ebony, or whitewash
3. **Low-profile:** Use 2x6 rails instead of 2x8 for lower profile
4. **Storage:** Add drawers underneath with casters
5. **LED lighting:** Route LED strips under the overhang for floating effect
