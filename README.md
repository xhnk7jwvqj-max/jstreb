# Trebuchet Designer

A web-based trebuchet simulation and design tool.

[Live Demo](https://hastingsgreer.github.io/jstreb)

## Development

```bash
yarn
npx vite
```

## Testing

### Unit Tests
```bash
yarn test
```

### E2E Tests

```bash
# Install Playwright browsers
npx playwright install chromium

# Run tests
yarn test:e2e

# Run with UI mode
yarn test:e2e:ui

# Debug mode
yarn test:e2e:debug
```
